# A.1 半导体物理原理

### **A.1.1 平衡状态下载流子浓度**

以四价元素硅为主，我们假设(除非另有说明)没有照明，没有辐射，没有机械应力，没有磁场，半导体的所有点都处于相同的温度(室温)，并且没有外部电压和电流施加到它上面。于是，半导体就被称为处于平衡状态。  

材料中有四种导电微粒，带正电微粒有空穴 \\(p\\) 和5价掺杂粒子 \\(N_D\\)（磷，其外围电子已被释放并成为自由电子，成为带正电粒子）。带负电微粒包括自由电子 \\(n\\) 与3价掺杂粒子\\(N_A\\)（硼，吸收一个电子成为带负电粒子）。因此电荷密度\\(\rho\\)可定义如下：

```katex
\rho = q(p+N_D-n-N_A)\tag{A-1}
```

平衡状态下半导体材料内部必须是中性（电荷密度处处为0）的，因此带电粒子浓度必须相同，即  

```katex
n-p=N_D-N_A \tag{A-2}
```

除此之外，空穴与电子浓度的乘积就是本征载流子浓度的平方  

```katex
np=n_i^2(T)\tag{A-3}
``` 

所以知道掺杂浓度，给定温度下可以直接求出\\(p\\)和\\(n\\)。但通常掺杂浓度足够大，n型掺杂中电子数会比空穴数大几个数量级。因此可默认浓度为

```katex
n \approx N_D, p \approx n_i^2(T)/N_D \tag{A-4}
```

**掺杂浓度很高**（大于\\(10^{18}cm^{-3}\\)）情况下不成立，半导体行为改变（费米能级进入导带/价带，玻尔兹曼统计不成立，用费米-狄拉克统计更新。被称为简并半导体）。此外**极低温**下掺杂原子不会被完全电离，或**温度升高**至\\(N_D \gg n_i\\)条件不成立时，上述近似也不成立。

以上是单种掺杂的情况。当p型参杂和n型掺杂相接触时，由于其内部电势不同，其内部会自发的形成电场。在半导体材料中，电流流动是两种效应的结果：载流子的**扩散**是由半导体中的**载流子浓度梯度**引起的，载流子的**漂移**是由**半导体中的电场**引起的。

如果半导体内部的电场不为零，则前面所给出的关系式一般不成立，\\(n\\)和\\(p\\)的值将与没有电场时的值不同。因此，考虑半导体材料(本征、p型或n型)处于平衡状态的区域，并假定两点\\(s\\)和\\(b\\)之间的静电电势差为\\(\phi_{sb}\\)。可以证明，两点的电子浓度\\(n_s\\)和\\(n_d\\)与静电电势差\\(\phi_{sb}\\)的关系如下：


```katex
n_s = n_be^{q\phi_{sb}/kT} \tag{A-5a}
```

空穴浓度与静电电势差\\(\phi_{sb}\\)的关系类似上述公式：

```katex
p_s = p_be^{-q\phi_{sb}/kT} \tag{A-5b}
```

上述两个（A-5）公式被成为**玻尔兹曼关系**。在电场存在的情况下，电荷密度\\(\rho\\)并不处处相等。但在平衡状态下公式（A-1）始终成立。此外，电荷密度还受**泊松分布**的约束，在单维度下其公式表示为：

```katex
\frac{d^2\phi}{dx^2}=-\frac{\rho(x)}{\varepsilon_s} \tag{A-6}
```


### **A.1.2 电流传输**
在平衡条件下，半导体中的移动载流子随机热运动不会产生平均电流。这种情况会可以被两种方式打破：
1. 施加了一个电场
2. 载流子分布不均匀

在这两种情况下，都会导致载流子的运动，从而产生电流。书中将这两种情况都视为单位横截面面积的一维，\\(x\\)在这里指的是所设计维度的距离/长度。符号上的下标\\(n\\)和\\(p\\)表示参数分别指n型或p型半导体；下标0的使用表示相应参数的热平衡值。这些符号表示在整本书中进行了统一。

##### **A.1.2.a 漂移电流**
电场在导体中的影响具有净效应，会沿场方向给予载流子加速度。这种净效应成为**漂移（drift）**，其会给载流子一个**漂移速度**，其中空穴和电子的漂移速度（\\(v_p/v_n\\)）公式如下：

```katex
v_p=\mu_pE \tag{A-7a}
```

```katex
v_n=-\mu_nE \tag{A-7b}
```

\\(\mu_p\\)和\\(\mu_n\\)分别是空穴和电子的迁移率。这里有假设迁移率和所施加的电场无关。这个假设一般而言是成立的，因为漂移速度通常会远远小于载流子的热运动（300K温度下硅的热运动速率为\\(10^7cm/s\\)）。在此基础上我们定义空穴与电子电流密度（\\(J_{p,drift}/J_{n,drift}\\)）：(Figure A-1 说明例外)

```katex
J_{p,drift}=qp\mu_pE \tag{A-8a}
```
```katex
J_{n,drift}=qn\mu_nE \tag{A-8b}
```

总漂移电流\\(J_{drift}\\)计算如下：

```katex
J_{drift}=q(p\mu_p+n\mu_n)E \tag{A-9}
```

半导体导电率\\(\sigma\\)就是\\(E\\)前的系数

```katex
\sigma=q(p\mu_p+n\mu_n) \tag{A-10}
```

##### **A.1.2.b 扩散电流**

**扩散（diffusion）**，是粒子随机热运动的一种表现形式，其表现为当载流子在系统中分布不均匀时出现的电流。扩散过程消除了半导体中的载流子浓度不平衡。与扩散相关的空穴/电子电流密度（\\(J_{p,diff}/J_{n,diff}\\)）公式如下：

```katex
J_{p,diff}=-qD_p\frac{dp}{dx} \tag{A-11a}
```
```katex
J_{n,diff}=qD_n\frac{dn}{dx} \tag{A-11b}
```

其中\\(D_n，D_p\\)分别是电子和空穴的扩散系数（这里的负号反了是因为求导下降方向带了一个符号）。这里同样假设其与\\(x\\)无关。

扩散的原理其实非常直观，跟载流子是否带电无关，仅仅是因为热运动导致的从高浓度区域朝向较低浓度区域的速度分量的载流子的数量大于具有相反方向的速度分量的载流子的数量（跟化学一样）。说人话就是高密度跑到低密度，跟总量没关系。总扩散电流公式如下：

```katex
J_{diff}=q(-D_p\frac{dp}{dx}+D_n\frac{dn}{dx})E \tag{A-12}
```

除电流特性分类以外，也可以根据电荷属性进行电流的分类，净电子电流与净空穴电流（\\(J_n/J_p\\)）计算公式如下：

```katex
J_{n}=q(n\mu_nE+D_n\frac{dn}{dx}) \tag{A-13b}
```
```katex
J_{p}=q(p\mu_pE-D_p\frac{dp}{dx}) \tag{A-13a}
```

**爱因斯坦关系**说明了漂移系数与扩散系数的关系，在这里直接给出公式，具体细节查阅文献：

```katex
\frac{D_p}{\mu_p}=\frac{D_n}{\mu_n}=\frac{kT}{q} \tag{A-14}
```

### **A.1.3 非平衡状态下半导体的载流子浓度**

然而，大部分半导体器件都工作在非平衡条件下（即正负载流子浓度积不同于其平衡值\\(n \times p \neq n_i^2\\)），**许多半导体器件的特性是由它们恢复平衡的倾向决定的**。非平衡状态下，公式（A-3）不成立，其对应两种情况：
```katex
np \gt n_i^{2} \quad 注入载流子 \quad
np \lt n_i^{2} \quad 抽取载流子  \tag{A-15}
```

首先我们定义**注入程度**概念：

1. **低注入程度**：注入的少数载流子浓度**远小于**平衡状态多数载流子浓度。
2. **高注入程度**：注入的少数载流子浓度**接近于**平衡状态多数载流子浓度。

这里的**注入**是一个瞬间的过程，对比的是注入少数载流子的变化浓度与原来平衡状态下的多数载流子浓度。注入瞬间便破坏了平衡状态，载流子就会试图恢复平衡，如果是*注入载流子*（一般而言对少数载流子影响更大），那么注入的载流子回合多数载流子*复合*来恢复平衡。如果是*抽取载流子*（一般而言对多数载流子影响更大），则是以*电子-空穴对*的产生来恢复平衡。

在实际中，超额载流子复合的平均时空速率与往往超额浓度近似成正比，低水平注入程度下也是如此。对p/n型半导体中的净复合速率可以以下公式给出：
```katex
R_p=\frac{p_n-p_{n0}}{\tau_p}=\frac{p^\prime_n}{\tau_p} \quad R_n=\frac{n_p-n_{p0}}{\tau_n}=\frac{n^\prime_p}{\tau_n}  \tag{A-16}
```
