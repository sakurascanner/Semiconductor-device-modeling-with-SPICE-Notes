# 1.1 DC电流-电压特性分析

### 1.1.1 理想二极管的直流特性
具体的二极管特性分析请见Appendix.2。正文中直接进行建模。(Figure 1-1 理想PN结正向偏置， Figure A-5 np区域浓度、电场、电势等变化)

总结理想二极管所做的假设：
1. p-和n-型掺杂区域都是均匀且电中性的，其浓度分别为\\(N_D\\)和\\(N_A\\)
2. 突变耗尽层近似（肖特基耗尽近似）：耗尽区（dipole layer，偶极层，指耗尽层正负电荷对峙的偶极结构）边界电压突变，而边界之外的半导体区域则是中性的。
3. 玻尔兹曼关系对整个耗尽区成立。
4. 低注入假设：注入的少数载流子密度比多数载流子密度小。
5. 在耗尽区中不存在产生电流，并且通过该区域的电子和空穴电流是恒定的。

这里直接给出理想二极管的电流公式：

$$
I_D=I_S(e^{qV_d/kT}-1) \tag{1-1}
$$
其中
```katex
I_S=qA_Jn_i^2\big(\frac{D_p}{N_DL_p}+\frac{D_n}{N_AL_n}\big) \tag{1-2a}
```
```katex
I_S=qA_Jn_i^2\big(\frac{D_n}{N_AW^\prime_p}+\frac{D_p}{N_DW^\prime_n}\big) \tag{1-2b}
```
分别是长基（\\(W^\prime \gg L\\)，少数载流子在扩散，到达接触点之前全部复合）二极管和短基二极管（\\(W^\prime \ll L\\)，少数载流子在复合之前到达接触点）的二极管反向饱和电流。其中\\(L_n\\)和\\(L_p\\)是电子和空穴的扩散长度；\\(W^\prime_n=(W_n-x_n)\\)和\\(W^\prime_p=(W_p-x_p)\\)则是**中性区（基区）**的长度，等于总长度减去耗尽层的长度。反向饱和电流为\\(I_S\\)。

在单边突变结（one-side abrupt junction，PN结一侧的掺杂浓度远远高于另一侧）的情况下，由于重掺杂侧的耗尽层宽度与掺杂浓度成反比（p重掺杂区域，\\(N_A\llN_D\\)为例，\\(x_p \propto 1/N_A\\)），此时重掺杂区的耗尽层几乎为0（\\(x_p\approx 0\\)），而轻掺杂区的耗尽层宽度几乎等于总耗尽层宽度（\\(x_n\approx W\\)）,\\(I_S\\)公式可简化为：
```katex
I_S\simeq\big(\frac{qA_Jn_i^2D_p}{N_DL_p}\big) \tag{1-3b}
```

```katex
I_S\simeq\big(\frac{qA_Jn_i^2D_p}{N_DW^\prime_n}\big) \tag{1-3b}
```

实际上，在\\(N_A \gg N_D\\)的情况下，p各点的浓度梯度也远远大于n（\\(dp_n/dx \gg dn_p/dx\\)），附录公式A-xx简化为：

```katex
I_D \simeq I_{p,diff}(X_n) \tag{1-4}
```
在极不对称的掺杂情况下，重掺杂区域通过扩散多数载流子保证了晶体管的电流通路，而少数载流子提供的电流可忽略不计。

### 1.1.2 理想二极管模型的限制

主要限制包含以下几点：

1. 耗尽区中的载流子生成-复合。
2. 高注入水平。
3. 与中性区的电场有关的压降。
4. 与高反向电压相关的内部击穿。

#### 1.1.2.a 载流子在耗尽区中的生成-复合。
(Figure 1-4 实际二极管的电流电压关系)

理想二极管中耗尽区不影响电流，但实际上耗尽区也会有生成-复合过程。与中性区不同，它是一个杂质梯度很大、载流子密度变化非常快的区域。耗尽区中载流子生成-复合的速率可以用Shockly-Hall-Read（SHR理论）来进行描述。其通用表达式为：

```katex
U=\frac{np-n_i^2}{\tau_p(n+n_1)+\tau_n(p+p_1)} \tag{1-5}
```

这里的\\(\tau_p,\tau_n,n_1,p_1\\)分别是空穴/电子的寿命以及陷阱能级相关的常数。正向偏置\\(V_D\gt 0\\)下，\\(np \gt n_i^2\\)意味着耗尽区发生载流子净复合损失电流；反向偏置\\(V_D\lt 0\\)下，\\(np \lt n_i^2\\)意味着耗尽区发生载流子净生成，导致了高于理想情况下的反向饱和电流。

个人理解：\\(np \gt n_i^2\\)时，电子-空穴相遇的可能增加，载流子复合而数量减少；\\(np \lt n_i^2\\)时，热运动生成的电子-空穴对来不及复合就被电场拉开，提供了额外的载流子。但结合和生成的变化速率跟电压的趋势不尽相同，如下所示：
```katex
I_{r,g}=qA_J\int_{-x_p}^{x_n}Udx=
\left\{
\begin{aligned}
%\nonumber
I_r&=f(e^{qV_d/2kT}), \quad \text{for} \quad V_D\gt 0  \\
I_g&=f(\sqrt{\frac{qV_D}{kT}}), \quad \text{for} \quad V_D\lt 0  \\
\end{aligned}

\tag{1-6}
\right.
```

在正向偏置\\(np \gg n_i^2, n=p\\)下，载流子浓度变化将远大于耗尽区宽度的变化，因此其内部电流近似由内**净复合率U**决定。在耗尽区中心处，净复合率可化简为：
```katex
U\approx \frac{n}{\tau_p+\tau_n} \tag{1-7}
```
而耗尽区中心的电子/空穴浓度与外加电压\\(V_D\\)的关系由结定律给出：
```katex
np=n_i^{2}exp(\frac{qV_D}{kT}) \tag{1-8}
```
因此\\(n=p\\)的情况下，其变化趋势为\\(e^{qV_d/2kT}\\)。

在反向偏置\\(np \ll n_i^2\\)下，净复合率U为常数，因此其电流与耗尽区的体积/宽度成正比。耗尽区的宽度\\(W\\)由下列公式给出：
```katex
W=\sqrt{\frac{2\epsilon_s(V_{bi}-V_D)}{qN_D}} \tag{1-9}
```
\\(V_{bi}\\)指内建电势。当反向偏置电压较大\\(|V_{D}| \gg V_{bi}\\)时，上述式子正比于\\(\frac{qV_D}{kT}\\)。

总而言之，一个经验公式可以用来有效描述上述特性：
```katex
I_D=I_S(e^{qV_D/nkT}-1) \tag{1-10}
```
这里的\\(n\\)被称作发射系数。复合电流主导的形况下其取值2，理想情况下取值1。

#### 1.1.2.b 高水平注入（定义见A.1.3）

低水平注入的情况下，少数载流子注入几乎不会影响多数载流子的浓度。但高水平注入的情况下则不是，浓度差会产生漂移电场，会导致结电压与\\(V_D\\)之间存在压降。以\\(p^+n\\)结为例，高水平注入时粒子浓度如下：
```katex
n_n=p_n+N_D \approx p_n \tag{1-10}
```
```katex
\frac{dn_n}{dx} \approx \frac{dp_n}{dx} \tag{1-11}
```
此外，\\(I_n\\)电流相较于\\(I_p\\)也可以忽略不计。因此根据公式（A-13b），其可认为：
```katex
I_{n}=qA_J(\mu_nn_nE+D_n\frac{dn}{dx})\approx 0 \tag{1-12}
```
解得电场\\(E\\)（使用了爱因斯坦关系）：
```katex
E=-\frac{D_n}{\mu_n}\frac{1}{n_n}\frac{dn_n}{dx}=-\frac{D_p}{\mu_p}\frac{1}{p_n}\frac{dp_n}{dx} \tag{1-13}
```

在高水平注入下，这里实际上没有满足中性区的假设。但由于高注入\\(n_n\\)浓度很高，所以电场实际上很小，因此，与中立条件的背离是可以忽略的，假设中立是可以接受的。知道电场的表达式后，我们就可以确定空穴电流：

```katex
I_{p}=qA_J(\mu_pp_nE-D_p\frac{dp_n}{dx})=-2qA_JD_p\frac{dp_n}{dx} \tag{1-14}
```

此公式说明了漂移电场的作用在高注入水平下和扩散作用是相同的，因此电流乘以二。如果是短基二极管且载流子浓度线性分布，则浓度梯度为常数：

```katex
\frac{dp_n}{dx}=const.=-\frac{p_n(x_n)}{W'_n} \tag{1-15}
```

根据方程（A-xxx A-74）以及高水平注入时的电荷中性条件，\\(p_n/x_n\\)的公式如下：
```katex
p_n(x_n)=n_n(x_n)=n_ie^{qV_D/2kT} \tag{1-16}
```

总结前面的公式，计算\\(I_D\\)可得：
```katex
I_D\approx I_p=\frac{2qA_JD_pn_i}{W'_n}e^{qV_D2kT} \tag{1-17}
```

#### 1.1.2.c 中性区电场导致的压降

在耗尽区中注入或提取多余的少数载流子总是伴随着中性区中的电场。因此，与该电场相关的中性区的压降与总电流成线性关系(仅在低水平注入和短基二极管前提下成立)。这些电压降可以通过对中性区上的电场\\(E\\)进行积分来评估。其中\\(V^\prime_D\\)是施加到二极管的电压，而\\(V_D\\)是跨越\\(pn\\)结的压降，则其关系为

```katex
V^\prime_D=V_D+\int_{Neutral Regions}{}(-E)dx \tag{1-18}
```

把电场的下降视为阻抗\\(r_S\\)，公式\\(1-18\\)可以从改写为：

```katex
V^\prime_D=\frac{kT}{q}ln(1+\frac{I_D}{I_S}) + r_SI_S \tag{1-19}
```
当\\(I_D\\)为小正数时，\\(V^\prime_D\\)与其对数相关；\\(I_D\\)为大正数时，则是线性相关。

#### 1.1.2.d 结击穿

反向电压超过耗尽区的电场临界值时，发生载流子倍增效应。其中包含**雪崩击穿**与**齐纳击穿**两种机制。

1. 雪崩击穿：热产生的载流子在耗尽区中的电场加速并获得动能，如果这个动能能够超过能带隙，当电子与晶格原子碰撞时，其失去的动能足以破坏共价键。因此除了自身之外，还产生了一个新的电子-空穴对。且这个过程可以重复，因此，每个新的载体又可能通过碰撞和断裂键的作用产生额外的载体。但这个过程**并非破坏性**的，比如IMPATT二极管和雪崩光电二极管是可以在雪崩击穿下工作的。*会导致损坏器件的是大量电流导致的热量积累*。

砷化镓和硅的雪崩击穿电场\\(E_{BR}\\)分别约为\\(3×10^5 V/cm\\)和\\(4×10^5 V/cm\\)。击穿电压\\(V_{BR}\\)可以以下公式给出：
```katex
|V_{BR}|=E_{BR}^2 \frac{\varepsilon_s}{2q}(\frac{1}{N_A}+\frac{1}{N_D}) \tag{1-20}
```
此公式说明了可以通过降低掺杂浓度中较小的一种以提高击穿电压（本质上是确保耗尽区的宽度足够大：\\(2E_{BR}W=|V_{BR}|\\)，低浓度耗尽区更长，压降分布更宽，电场更小）。

2. 齐纳击穿：可能出现在结的两个区域都有高掺杂浓度的反向偏置pn结中。较高的掺杂浓度保证了耗尽区宽度较小。发生齐纳击穿的电压通常在3-6V附近，通常远低于发生雪崩击穿时的电压。另外，如果掺杂浓度使得零偏压耗尽区宽度较大，则可能不可能增加对电压的反向偏置，从而获得用于隧道传输的必要的物理性质，因为雪崩击穿将首先发生。关于雪崩击穿，齐纳击穿不像雪崩击穿那样涉及载流子与晶格原子的碰撞；此外，齐纳击穿中的电导\\(dI_D/dV_D\\)很高，齐纳二极管适合用作稳压电路。
