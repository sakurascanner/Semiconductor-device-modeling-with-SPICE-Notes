# 1.2 静态模型

### **1.2.1 理想二极管的静态模型以及其SPICE2实现**

#### **1.2.1.a 正向直流特性**

理想二极管的正向直流特性由以下非线性电流源\\(I_D\\)来表示:

```katex
I_D = I_S(e^{qV_D/kT}-1) + V_D \text{GMIN \\  for } V_D \gt -5\frac{kT}{q} \quad \tag{1-21}
```

这里的\\(V_D\\)指的是被施加在结上的电压而非施加给二极管的电压。其模型仅有一个参数：**饱和电流**\\(I_S\\)。一旦通过.MODEL指定就作为基础参数进行计算。

公式中还有**GMIN**项。此项的作用是用于帮助收敛，公式中体现为添加了一个极小的电导（程序默认\\(10^{-12}S\\)），可被.OPTION语句修改但不可为零。


#### **1.2.1.a 反向直流特性**
理想二极管的反向直流特性由以下非线性电流源\\(I_D\\)来表示:

```katex
I_D=f(V_D)=
\left\{
\begin{aligned}
&I_S(e^{qV_D/kT}-1)+V_D \text{GMIN}   &&\text{for} -5\frac{kT}{q} \leq V_D \leq 0 \\
&-I_S+V_D \text{GMIN}   &&  \text{for} \quad \quad V_D \lt -5\frac{kT}{q} \\
\end{aligned}

\tag{1-22}
\right.
```

### **1.2.2 实际二极管的静态模型以及其SPICE2实现**

#### **1.2.1.a 微偏置：载流子复合-生成效应**

考虑载流子复合-生成效应，二极管直流特性实际为（多了个n，发射系数）：
```katex
I_D=f(V_D)=
\left\{
\begin{aligned}
&I_S(e^{qV_D/nkT}-1)+V_D \text{GMIN}   &&\text{for}  \quad  V_D \geq -5\frac{nkT}{q} \\
&-I_S+V_D \text{GMIN}   &&  \text{for} \quad V_D \lt -5\frac{nkT}{q} \\
\end{aligned}

\tag{1-23}
\right.
```

这里的n具体值可以由用户指定。指定后不可被修改。

#### **1.2.1.b 高偏置：欧姆电阻和高水平注入的影响**

高偏置下须考虑中性区和触点的电阻，以及高水平注入的等效阻抗。在SPICE2的模型实现中并没有对这些效应进行直接建模，而是由一个欧姆电子\\(r_s\\)来模拟，相当于在二极管两端内包含一个理想二极管串联一个\\(r_s\\)，则电压计算公式如下：

```katex
V^\prime_D=r_SI_D+V_D \tag{1-24}
```

#### **1.2.1.c 高反向偏置：结击穿的影响**

*书中没给具体原理，直接给的图例和公式，估计是超纲了。。这里需要计算倍增因子\\(M\\)，其需要求解玻尔兹曼输运方程或电离积分以及电离系数（Chynoweth 定律），并且需要基于实验数据拟合。这里先省略。*

综合考虑以上情况后，反向偏置时的\\(I_D\\)可以被表示为：

```katex
I_D=f(V_D)=
\left\{
\begin{aligned}
&I_S(e^{qV_D/nkT}-1)+V_D \text{GMIN}   &&\text{for}  \quad -5 \frac{nkT}{q} \leq V_D \leq 0\\
&-I_S+V_D \text{GMIN}   &&\text{for} \quad -\text{BV} \lt V_D \lt -5\frac{nkT}{q} \\
&-\text{IBV}   &&\text{for}  \quad  V_D = -\text{BV} \\
&-I_S(e^{-q(BV+V_D)/kT}-1+\frac{q\text{BV}}{kT})   &&\text{for}  \quad  V_D \lt -\text{BV} \\
\end{aligned}

\tag{1-25}
\right.
```

为了保证求解器的收敛性，上述公式保证了电压值为\\(V_D=\text{BV}\\)时的数值相等。

二极管正向和反向特性(二极管静态模型)在SPICE2中通过以下参数建模，用户可以在.MODEL语句中指定这些参数：

```katex
\begin{aligned}
&IS   &&\text{saturation current}(I_S)\text{，饱和电流}\\
&RS   &&\text{ohmic resistance}(r_S)\text{，欧姆电阻}\\
&N    &&\text{emission coefficient}(n)\text{，发射系数}\\
&BV   &&\text{breakdown voltage (BV) (it is a positive number，击穿电压，正值)}\\
&IBV  &&\text{breakdown current (IBV) (it is a positive number，击穿电流，正值)}\\
\end{aligned}
```

图（Figure 1-14）可用于描述齐纳二极管的宏模型：


Laha和Smarte二人在SPICE2中以以下公式描述了齐纳二极管：

```katex
I_D=f(V_D)=
\left\{
\begin{aligned}
&I_S(e^{qV_D/nkT}-1)                                            &&\text{for} \quad V_D \gt 0\\
&\frac{qV_D}{nkT}                                               &&\text{for} \quad -n\frac{kT}{q} \leq V_D \leq 0 \\
&-I_S                                                           &&\text{for} \quad -\text{BV} \lt V_D \lt -n\frac{kT}{q} \\
&-e^{-K_1(V_1+V_D)}--e^{-K_2(V_2+V_D)}+e^{K_1V_1}+e^{K_2V_2}    &&\text{for} \quad  V_D \leq -\text{BV} \\
\end{aligned}

\tag{1-26}
\right.
```

这里的\\(K_1,V_1,K_2,V_2\\)数值选取必须小心保证其连续。如果\\(BV \gt -nkT/q\\)，则直接从\\(V_D=0\\)开始采用击穿公式。书中还提到有其他的极限算法，不懂。
