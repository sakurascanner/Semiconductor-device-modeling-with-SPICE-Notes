# 1.3 大信号模型

### **1.3.1 二极管大信号模型**

大信号模型主要专注于电荷存储。没有电荷存储，理论上充放电就会无限快，而这是不可能的。二极管主要有由于掺杂浓度而在耗尽区的电荷存储以及由于注入中性区域的少数载流子电荷而产生的电荷存储：

```katex
Q^\prime_j=\sqrt{\frac{2q\varepsilon_s(\phi_0-V_D)}{\frac{1}{N_A}+\frac{1}{N_D}}} \tag{1-27}
```
```katex
Q^\prime_d=Q^\prime_p+Q^\prime_n=\tau_DI_D \tag{1-28}
```

其中\\(\phi_0\\)是内建电势，\\(Q^\prime_p,Q^\prime_n\\)是单位面积电荷分别由存储在n型区域中的空穴和存储在p型区域中的电子产生；\\(\tau_D\\)是二极管传输时间，这两种电荷在单位面积上都产生两个小信号电容，通常分别称为结电容\\(C^\prime_j\\)和扩散电容\\(C^\prime_d\\)。

参考A.2.4节，结电容\\(C^\prime_j\\)公式如下：

```katex
C^\prime_j=\frac{dQ^\prime_j}{dV_D}=\sqrt{\frac{q\varepsilon_s}{2(\phi_0-V_D)(\frac{1}{N_A}+\frac{1}{N_D})}}=\frac{C^\prime_j(0)}{\sqrt{1-\frac{V_D}{\phi_0}}} \tag{1-29}
```

其中\\(C^\prime_j(0)\\)是平衡时(即\\(V_D=0\\)时)的单位面积电容。扩散电容\\(C^\prime_d\\)公式如下：

```katex
C^\prime_d=\frac{dQ^\prime_d}{dV_D}=\frac{q}{nkT}\tau_DI_Se^{qV_D/nkT} \tag{1-30}
```

公式（1-27）也可以被写为0偏置下的电荷总量：

```katex
Q^\prime_j=2\phi_0C^\prime_j(0)\sqrt{1-\frac{V_D}{\phi_0}} \tag{1-31}
```

总电荷、电容可用上面两电容相加：
```katex
Q^\prime_D=Q^\prime_j+Q^\prime_d \tag{1-32}
```
```katex
C^\prime_D=\frac{dQ^\prime_D}{dV_D}=\frac{Q^\prime_j+Q^\prime_d}{dV_D} \tag{1-33}
```

SPICE2中二极管的大信号模型图（Figure 1-15）对于反向偏置和微正向偏置，\\(Q^\prime_j\\)是主要的电荷存储方式。对于其他情况，注入电荷\\(Q^\prime_D\\)占主导地位，因此\\(Q^\prime_j\\)的影响可以忽略不计。
