# 1.3 大信号模型

### **1.3.1 二极管大信号模型**

大信号模型主要专注于电荷存储。没有电荷存储，理论上充放电就会无限快，而这是不可能的。二极管主要有由于掺杂浓度而在耗尽区的电荷存储以及由于注入中性区域的少数载流子电荷而产生的电荷存储：

```katex
Q^\prime_j=\sqrt{\frac{2q\varepsilon_s(\phi_0-V_D)}{\frac{1}{N_A}+\frac{1}{N_D}}} \tag{1-27}
```
```katex
Q^\prime_d=Q^\prime_p+Q^\prime_n=\tau_DI_D \tag{1-28}
```

其中\\(\phi_0\\)是内建电势，\\(Q^\prime_p,Q^\prime_n\\)是单位面积电荷分别由存储在n型区域中的空穴和存储在p型区域中的电子产生；\\(\tau_D\\)是二极管传输时间，这两种电荷在单位面积上都产生两个小信号电容，通常分别称为结电容\\(C^\prime_j\\)和扩散电容\\(C^\prime_d\\)。

参考A.2.4节，结电容\\(C^\prime_j\\)公式如下：

```katex
C^\prime_j=\frac{dQ^\prime_j}{dV_D}=\sqrt{\frac{q\varepsilon_s}{2(\phi_0-V_D)(\frac{1}{N_A}+\frac{1}{N_D})}}=\frac{C^\prime_j(0)}{\sqrt{1-\frac{V_D}{\phi_0}}} \tag{1-29}
```

其中\\(C^\prime_j(0)\\)是平衡时(即\\(V_D=0\\)时)的单位面积电容。扩散电容\\(C^\prime_d\\)公式如下：

```katex
C^\prime_d=\frac{dQ^\prime_d}{dV_D}=\frac{q}{nkT}\tau_DI_Se^{qV_D/nkT} \tag{1-30}
```

公式（1-27）也可以被写为0偏置下的电荷总量：

```katex
Q^\prime_j=2\phi_0C^\prime_j(0)\sqrt{1-\frac{V_D}{\phi_0}} \tag{1-31}
```

总电荷、电容可用上面两电容相加：
```katex
Q^\prime_D=Q^\prime_j+Q^\prime_d \tag{1-32}
```
```katex
C^\prime_D=\frac{dQ^\prime_D}{dV_D}=\frac{Q^\prime_j+Q^\prime_d}{dV_D} \tag{1-33}
```

SPICE2中二极管的大信号模型图（Figure 1-15）对于反向偏置和微正向偏置，\\(Q^\prime_j\\)是主要的电荷存储方式。对于其他情况，注入电荷\\(Q^\prime_D\\)占主导地位，因此\\(Q^\prime_j\\)的影响可以忽略不计。特别是当\\(V_D\to\\phi_0\\)时，结电容趋近于无穷大而扩散电容近乎为0（当然电压大到这种程度公式（1-30）也不成立了）。结电容与电压的关系如图（Figure 1-16）所示。可以看到当\\(\phi_0/2 \lt C_j^\prime\lt \phi_0\\)时，公式（1-30）基本成立。


### **1.3.2 二极管大信号模型及SPICE仿真**

我们首先给出总电荷\\(Q_D=Q_j+Q_d\\)的实现公式：

```katex
Q_D=
\left\{
\begin{aligned}
&\underbrace{\tau_DI_D}_{Q_d}+\underbrace{C_j(0)\int_{0}^{V_D}(1-\frac{V}{\phi_0}^-m)dV}_{Q_j}  \quad &&\text{for} \quad V_D \lt FC \times \phi_0 \\
&\underbrace{\tau_DI_D}_{Q_d}+\underbrace{C_j(0)F_1+\frac{C_j(0)}{F_2}\int_{FC\times\phi_0}^{V_D}\left(F_3+\frac{mV}{\phi_0}\right)dV}_{Q_j} 
&&\text{for} \quad V_D \lt FC \times \phi_0 
\end{aligned}
\tag{1-34}
\right.
```

同理，电容-电压关系可得：

```katex
C_D=\frac{dQ_D}{dV_D}=
\left\{
\begin{aligned}
&\underbrace{\tau_D\frac{dI_D}{dV_D}}_{C_d}+\underbrace{C_j(0)\left(1-\frac{V}{\phi_0}\right)^{-m}}_{C_j}  \quad &&\text{for} \quad V_D \lt FC \times \phi_0 \\
&\underbrace{\tau_D\frac{dI_D}{dV_D}}_{C_d}+\underbrace{\frac{C_j(0)}{F_2}\left(F_3+\frac{mV}{\phi_0}\right)}_{C_j} 
&&\text{for} \quad V_D \lt FC \times \phi_0 
\end{aligned}
\tag{1-35}
\right.
```

其中\\(F_1, F_2, F_3\\)公式如下：

```katex
\begin{aligned}
F_1 &= \frac{\phi_0}{1-m}\left[1-(1-FC)^{1-M}\right] \\
F_2 &= (1-FC)^{1+M}\\
F_3 &= 1-FC(1+M)
\end{aligned} \tag{1-36}
```

以上公式可以总结，因少数载流子注入导致的扩散电容可以由反向饱和电流\\(I_S\\)，发射系数\\(n\\)（理想二极管默认取1），以及载流子转移时间\\(\tau_D\\)。并且由于\\(Q_d\propto I_D\\)，其与实际电压在一定范围内成指数相关。

结电荷则由\\(C_D(0)\\)，二极管在未偏置的情况下的电容；\\(m\\)，分级系数；结电势\\(\phi_0\\)以及\\(FC\\)，正向偏置结电容系数来共同决定。其中结电势\\(\phi_0\\)大约在0.2到1伏之间，分级系数\\(m\\)对线性渐变结可取值0.33，突变结取0.5。\\(FC\\)系数取值范围为0到1之间，默认0.5，其决定了在结正向偏置时如何计算结电容。

SPICE2中用于描述二极管的电容参数包含：
```katex
\begin{aligned}
&TT   &&\text{Transit time}(\tau_D)\text{，转移时间}\\
&CJO   &&\text{Zero-bias junction capacitance}(C_j(0))\text{，零偏置结电容}\\
&M    &&\text{Grading coefficient}(m)\text{，分级系数}\\
&VJ   &&\text{Junction potential}(\phi_0)\text{，结电势}\\
&FC  &&\text{Coefficient for forward-bias depletion capacitance formula}(FC)\text{，正向偏置耗尽区电容系数}\\
\end{aligned}
```
