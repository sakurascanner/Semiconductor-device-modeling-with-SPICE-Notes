# 1.5 肖特基势垒二极管（SBD）以及SPICE实现。

当金属与半导体(例如，n型衬底)接触时，通常形成整流势垒，被称为肖特基势垒二极管。附录C中描述的这些结构在电气上类似于突然的单边p+n结。但相比于p+n结其具有两个优势：

1. 首先，SBD在低水平注入条件下作为多数载流子器件运行。结果，消除了(由于存储少数载波)的存储时间，并且获得了快速响应。
2. 其次，对于固定的偏置电压，流过器件的电流通常比结器件的电流大两个数量级以上，如图1-19所示。

SBD的电流特性如下：
```katex
I_D=I_S(e^{qV_D^\star/nkT}-1)

\tag{1-40}
```

其中\\(I_S\\)公式如下：
```katex
I_S=A^\star T^2e^{-q\phi_B/kT}
\tag{1-41}
```
这里的\\(\phi_B\\)是肖特基势垒电压，其取决于所使用的金属半导体材料。\\(A^\star\\)是常数，被称为有效理查森常数（effective Richardson constant）。其与温度无关，但它取决于所考虑的特定半导体和晶体朝向。对N型硅，\\(A^\star=250A\cdot cm^{-2}\cdot K^{-1}\\);对N型砷化镓，\\(A^\star=8.1\cdot cm^{-2}\cdot K^{-1}\\)。

轻掺杂衬底上的肖特基二极管，其反向击穿机制与PN结相同，均为雪崩击穿，但击穿电压通常更低；而在重掺杂衬底上，则可能通过隧穿效应发生击穿，其类似于PN结的齐纳击穿。

对于大信号模型，由于少数载流子注入和存储到SBD的半导体中的次数很少，少数载流子的电荷存储效应几乎完全由公式(A-51.5)给出的结电容决定。为方便起见，在此重写一遍：
```katex
C^\prime_J=\frac{\varepsilon_s}{W} \tag{A-51.5}
```
其中W是耗尽区宽度，公式如下：
```katex
W=\sqrt{\frac{2\varepsilon_s}{qN_D}(\phi_{s0}-V_D)} \tag{1-42}
```
其中势能\\(phi_{s0}\\)表示金属中的电子相对于半导体中的电子的能量平衡差：
```katex
\phi_{s0}=\phi_B-\phi_n \tag{A-51.5}
```
其中\\(\phi_n\\)由公式（A-21.1）给出，电势\\(phi_{s0}\\)的作用类似于\\(phi_{0}\\)在PN结中的作用。

因此，SBD通常被称为多数载流子器件，即这些器件中的正向偏置电流是通过从n型半导体向金属发射电子来表征的，而不是通过向半导体中注入空穴来表征的。因此，通常可以忽略SBD中的少数载流子电荷存储效应。

SBD的小信号模型也只包括差动电导\\(G_D\\)电阻以及与之并联的结电容\\(C_j\\)。

**实际上，SPICE2中没有明确实现肖特基二极管的模型；但是，可以使用SPICE2中的pn结二极管模型进行模拟，方法是将少数载流子存储时间\\(TT\\)设置为零，并通过公式（1-42）获得饱和电流值。将内置电压\\(VJ\\) 设为 \\(\phi_{s0}\\)，为为零偏置结电容\\(CJO\\)选择合适的值，并将发射系数N设置为1，将分级系数M设置为0.5。**

基于以上设置，PN结二极管的三种信号模型也同样可用在肖特基二极管中。（Figure 1-9,15,18）
