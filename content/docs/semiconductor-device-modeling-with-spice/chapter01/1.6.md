# 1.6 温度和面积对二极管模型参数的影响
首先简要介绍显著依赖于温度的物理量。

### **1.6.1 二极管特性的温度依赖特性**
虽然在二极管的方程中，指数项已经包含了绝对温度\\(qV_D/kT\\)，但实际上载流子浓度随温度的影响被我们忽略了。而其会严重影响饱和电流\\(I_S\\)以及内置电压\\(\phi_0\\)的计算。载流子总浓度随温度的变化公式如下：
```katex
n_i^2(T)=BT^3e^{-E_g(T)/kT}
\tag{1-40}
```
其中\\(B\\)是常数，\\(E_g\\)是温度的函数，公式如下：
```katex
E_g(T)=E_g(0)-\frac{\alpha T^2}{\beta+T}   \tag{1-41}
```

以硅为例，其参数为\\(\alpha=7.02\times 10^{-4}, \beta=1108\\)，\\(E_g(0)=1.16eV\\)。


### **1.6.2 SPICE2二极管模型参数的温度依赖性**

SPICE2中.MODEL语句的所有输入数据都是在27°C(300 K)下测量的。仿真还假设名义温度(TNOM)为27摄氏度，可通过.OPTIONS控制语句的TNOM选项进行更改。

通过使用.TEMP控制语句，可以在TNOM以外的温度下进行电路仿真。温度显式地出现在二极管模型方程的指数项中。另外，SPICE2还修改了四个二极管参数以反映pn结和SBD二极管的温度变化，其包括饱和电流\\(I_S\\)，结电势\\(\phi_0\\)，零偏结电容\\(C_D(0)\\)，正向偏置损耗电容系数\\(FC\\)。

\\(I_S\\)更新公式如下：
```katex
I_S(T_2)=I_S(T_1)\left(\frac{T_2}{T_1}\right)^{\text{XT1}/n}\text{exp}\left[-\frac{qE_g(300)}{kT_2}\left(1-\frac{T_2}{T_1}\right)\right]
\tag{1-42}
```
这里引入了两个新的模型参数，\\(\text{XTI}\\)，饱和电流温度指数；\\(E_g\\)，能隙。pn结和SBD对以上参数的默认值是不同的。\\(\text{XTI} = 3\\)对pn结，\\(\text{XTI} = 2\\)对SBD；\\(E_g(300K) = 1.11eV\\)对pn结，\\(E_g(300K) = 0.69eV\\)对SBD。

\\(\phi_0\\)更新公式如下：
```katex
\phi_0(T_2)=\frac{T_2}{T_1}\phi_0(T_1)-2\frac{kT_2}{q}\ln\left(\frac{T_2}{T_1}\right)^{1.5}-\left[\frac{T_2}{T_1}E_g(T_1)-E_g(T_2)\right]  \tag{1-43}
```

\\(C_D(0)\\)更新公式如下：
```katex
C_D(T_2)=C_D(T_1)\left\{1+m\left[400 \times 10^{-6}(T_2-T_1)-\frac{\phi_0(T_2)-\phi_0(T_1)}{\phi_0(T_1)}\right]\right\}   \tag{1-44}
```

\\(FC\\)更新公式如下：
```katex
FCPB(T_2)=\text{FC}\times \phi_0=FCPB(T_1)\frac{\phi_0(T_2)}{\phi_0(T_1)}
F_1(T_2)=F_1(T_1)\frac{\phi_0(T_2)}{\phi_0(T_1)}  \tag{1-45}
```

上述公式中的\\(T_1,T_2\\)在程序运行中的情况如下：
1. 如果\\(TNOM\\)没有指定， 那么默认300K；
2. 如果在.OPTION中指定了\\(TNOM_{new}\\)，那么将300K的数值作为T1，\\(TNOM_{new}\\)的值作为T2。
3. 如果指定了一系列的\\(TNOM_{new}\\)，则按堆栈顺序进行计算仿真。


### **1.6.3 SPICE2二极管模型参数的面积影响**

面积可以视为多个器件平行。因此参数变化如下：
```katex
I_S=I_S\times \text{AREA}  
C_D(0)=C_D(0)\times \text{AREA}  
r_S=r_S\times \text{AREA}  
\tag{1-46}
```
