# 1.8 HSPICE模型

HSPICE的二极管模型包括结二极管模型和Fowler-Nordheim模型。

结二极管模型有两种变体：几何模型和非几何模型（Geometric Model 是“版图驱动”的模型；Nongeometric Model 是“数据手册驱动”的模型，没有几何尺寸数据。）。几何结二极管用于模拟基于IC的硅扩散二极管、肖特基势垒二极管和齐纳二极管。几何模型允许为特定的IC工艺和技术指定pn结、多晶硅和金属电容尺寸。非几何结二极管用于模拟离散二极管器件，如标准二极管和齐纳二极管。非几何模型允许通过使用无量纲面积参数来缩放电流、电阻和电容。

Fowler-Nordheim二极管定义了流经绝缘体的隧道电流(见第1.8.6节)。其用于对非易失性EEPROM存储器中的二极管效应进行建模。

.MODEL语句中的Level参数选择使用的二极管型号。特别是，HSPICE的用户必须为非几何结二极管模型设置Level=1，为Fowler-Nordheim二极管模型设置Level=2，为几何结二极管模型设置Level=3。如果省略了LEVEL参数，则模型默认为LEVEL=1。如果在.MODEL语句中作为模型参数重复使用元素参数，则优先于模型参数。ELEMENT(D)和.MODEL语句共有的参数有：Area、PJ、M、LM、LP、WM、WP、W和L(参见第节。1.8.5)。

### **1.8.1 结二极管静态模型（LEVEL=1,3）**

HSPICE中也把二极管分为了三个区域：正向偏置、反向偏置、击穿。

之前分析二极管模型得到的理想指数方程也被HSPICE用来定义正向和反向区域的静态模型：

```katex
I_D=
\left\{
\begin{aligned}
&\frac{I^\star_D}{1+\left(\frac{I_D^\star}{I_{KF}}\right)^{1/2}}  \quad &&\text{for} \quad V_D \geq -10\frac{nkT}{q} \\
&\frac{I^\star_D}{1+\left(\frac{I_D^\star}{I_{KR}}\right)^{1/2}}
&&\text{for} \quad V_D \lt -10\frac{nkT}{q}
\end{aligned}
\tag{1-47}
\right.
```

其中\\(I_D^\star\\)被定义为

```katex
I_D^\star=
\left\{
\begin{aligned}
&I_S(e^{qV_D/NKt}-1)  &&\text{for} \quad V_D \geq -10\frac{nkT}{q} \\
&I_S(e^{qV_D/NKt}-1)-I_S[e^{-q(\text{BV}+V_D)/nkT-1}]
&&\text{for} \quad V_D \lt -10\frac{nkT}{q}
\end{aligned}
\tag{1-48}
\right.
```

在反向偏置区域中，可以以\\(\text{BV}\\)为分界，进一步区分其行为：

```katex
I_D^\star=
\left\{
\begin{aligned}
&-I_S  &&\text{for} \quad -\text{BV} \lt V_D \lt -10\frac{nkT}{q} \\
&-I_Se^{-q(\text{BV}+V_D)/nkT}
&&\text{for} \quad V_D \lt -\text{BV}
\end{aligned}
\tag{1-49}
\right.
```

在.MODEL状态中指定的反向击穿电压\\(\text{BV}\\)调整如下：定义\\(I_{break}\\)：
```katex
I_{break}=-I_S(e^{-q\text{BV}/nkT}-1)
\tag{1-50}
```

并且.MODEL中的参数\\(\text{IBV}\\)，击穿电压时的电流大于\\(I_{break}\\)，那么\\(\text{BV}\\)可描述为：
```katex
\text{BV}_{adjusted}=\text{BV}-\frac{nkT}{q}\ln\left(\frac{\text{IBV}}{I_{break}}\right)
\tag{1-51}
```
如果没有大于，那么令：

```katex
\text{IBV}=I_{break}
\tag{1-52}
```

从上面的公式中可以看到，HSPICE相比SPICE2引入了两个新参数：\\(\text{IKF}(I_{KF})\\)，正向膝电流和\\(\text{IKR}(I_{KR})\\)，反向膝电流，其分别对正/反高掺杂效应进行了建模。（见章节1.1.2b和1.2.2b）其他的参数比如\\(\left[\text{IS}(I_S), \text{N}(n), \text{BV}, \text{IBV}, \text{RS}(r_S)\right]\\)仍然保持一致。如果\\(\text{IKF}(I_{KF})\\)和\\(\text{IKR}(I_{KR})\\)没有被指定，那么HSPICE认为\\(I_D=I_D^\prime\\)，整个关系和SPICE2保持一致。还有决定正反偏置区域边界的热电压值不同，HSPICE\\(\to -10nkT/q\\)而SPICE\\(\to -5nkT/q\\)。


### **1.8.2 结二极管大信号模型（LEVEL=1,3）**

HSPICE中也用\\(C_D\\)来表示电容。但是这里的电容包含了以下电容：扩散电容\\(C_d\\)，结电容\\(C_j\\)，金属电容\\(C_m\\)和多晶硅电容\\(C_p\\)。
```katex
C_D=C_d+C_j+C_m+C_p 
\tag{1-53}
```

其中扩散电容\\(C_d\\)和SPICE中的基本相同，见公式（1-30）。因此其也由转移时间\\(\text{TT}\\)来描述。

结电容\\(C_j\\)则分为结底部电容\\(C_{ja}\\)和结周边电容\\(C_{jp}\\)两部分。这两个电容的计算公式比较相似，除了有不同的模型参数。其中结底部电容\\(C_{ja}\\)等于SPICE中的\\(C_{j}\\)，见公式（1-35的第二项）。因此HSPICE中的参数包括\\([CJO(CJ), VJ(PB), M(MJ), FC]\\)。括号中显示的是可以在HSPICE中使用的其他等价关键字。具体对应关系请见表1-1。

结周边电容\\(C_{jp}\\)由以下公式描述：

```katex
C_{jp}=
\left\{
\begin{aligned}
&C_{jp}(0)\left(1-\frac{V_D}{\phi_{0p}}\right)^{-m_{jsw}}  \quad &&\text{for} \quad V_D \lt FCS \times \phi_{0p} \\
&\frac{C_{jp}(0)}{F_2}\left(F_3+\frac{m_{jsw}V_D}{\phi_{0p}}\right)
&&\text{for} \quad V_D \geq FCS \times \phi_{0p}
\end{aligned}
\tag{1-54}
\right.
```

其中包含了四个新参数：\\(\text{CJP}(C_{jp})\\)，零偏周边电容；\\(\text{PHP}(\phi_{0p})\\)，外围结接触点电势；\\(\text{MJSW}(m_{jsw})\\)，周边结点分级系数；\\(\text{FCS}\\)，正向偏置耗尽外围电容公式的系数。其中的\\(F_2,F_3\\)参数遵循公式（1-36），而\\(m\\)和\\(\text{FC}\\)分别由\\(\text{MJSW}(m_{jsw})\\)和\\(\text{FCS}\\)替代。总结电容由上述两个分电容相加：

```katex
C_j=C_{ja}+C_{jp}
\tag{1-55}
```

多晶硅电容\\(C_p\\)和金属电容\\(C_m\\)可以用以下公式说明：

```katex
C_p=\frac{\varepsilon_{ox}}{X_{P0}}(W_P+X_P)(L_P+X_P)M
\tag{1-56}
```

```katex
C_m=\frac{\varepsilon_{ox}}{X_{M0}}(W_M+X_M)(L_M+X_M)M
\tag{1-57}
```

其中\\(\varepsilon_{ox}\\)是氧化物介电常数。这两个公式仅限于\\(LEVEL=3\\)中且用户必须在.MODEL中声明参数。需要声明的参数包括：\\(\text{XOI}(X_{P0})\\)，多晶硅与体氧化层的厚度；\\(\text{WP}(W_{P})\\)，多晶硅的宽度；\\(\text{XP}(X_{P})\\)，掩模和蚀刻效果；\\(\text{LP}(L_{P})\\)，多晶硅的长度；\\(\text{XOM}(X_{M0})\\)，金属到块状氧化物的厚度；\\(\text{WM}(W_{M})\\)，金属层的宽度；\\(\text{XM}(X_{M})\\)，掩蔽和蚀刻效果；\\(\text{LM}(L_{M})\\)，金属的长度。\\(\text{M}\\)，模拟多个二极管的乘数系数。


### **1.8.3 结二极管小信号模型（LEVEL=1,3）**

如同在SPICE2中一样，HSPICE小信号模型通过静态和大信号模型方程的线性化在程序中自动计算(见章节1.4.1)；因此，HSPICE的用户不需要在.MODEL语句中指定新的模型参数。

### **1.8.4 温度对二极管模型参数的影响（LEVEL=1,3）**

LEVEL1,3二极管模型都包含参数\\(\text{TLEV}\\)和\\(\text{TLEVC}\\)，以选择不同的温度方程来计算温度对能隙、饱和电流、击穿电压、渡越时间、接触电势、结电容、分级系数和欧姆串联电阻的影响。后面的推导仅考虑当\\(\text{TLEV}\\)和\\(\text{TLEVC}\\)为0的时候的方程。

HSPICE同样遵循在SPICE2中能隙与温度的关系，如公式（1-41）。而饱和电流的温度依赖关系\\(IS(I_S)\\)由公式（1-42）确定。

击穿电压\\(\text{BV}\\)与温度的关系：
```katex
\text{BV}(T_2)=\text{BV}(T_1)-\text{TCV}(T_2-T_1)
\tag{1-58}
```
其中\\(\text{TCV}\\)是击穿电压温度系数，需通过HSPICE中的.MODEL语句指定。

转移时间\\(\tau_D(\text{TT})\\)与温度的关系：
```katex
\text{TT}(T_2)=\text{TT}(T_1)[1+\text{TTT1}(T_2-T_1)+\text{TTT2}(T_2-T_1)^2]
\tag{1-59}
```
其中\\(\text{TTT1}, \text{TTT2}\\)是一阶和二阶温度系数，同样需要用户指定。

和SPICE2一样，HSPICE中的面结接触电势\\(\phi_0 \text{PB}\\)通过方程（1-49）与温度相关。类似的关系对HSPICE的参数\\(\phi_{0p} \text(PHP)\\)，外围结接触电势同样适用。

单位结低部面积的偏置结电容\\(C_j(0)，\text{CJCO}\\)遵循公式（A-50）;而单位结外围的零偏置结电容\\(C_{jsw}，\text{CJSW}\\)公式如下：
```katex
\text{TT}(T_2)=\text{TT}(T_1)[1+\text{TTT1}(T_2-T_1)+\text{TTT2}(T_2-T_1)^2]
\tag{1-59}
```
